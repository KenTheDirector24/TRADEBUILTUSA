# TradeBuilt

A modern home for the skilled trades — electrical, plumbing, HVAC, welding, carpentry, and more. Built as a personal passion project, growing one feature at a time.

## Running locally

No build step required. Serve the folder with any static server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
